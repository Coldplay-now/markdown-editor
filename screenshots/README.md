# 截图目录

请将以下截图文件放置在此目录：

1. `screenshot-1.png` - 主界面截图（显示代码高亮、数学公式、表格等功能）
2. `screenshot-2.png` - Mermaid 图表截图（显示流程图、序列图、饼图等）

## 如何添加截图

### 方法 1：手动添加
1. 运行应用程序 `npm run dev`
2. 使用截图工具截取应用界面
3. 将截图保存为 `screenshot-1.png` 和 `screenshot-2.png`
4. 放置在此目录中

### 方法 2：从现有图片复制
如果您已经有截图，请将它们复制到此目录并重命名为：
- `screenshot-1.png`
- `screenshot-2.png`

## 截图规范

- **格式**：PNG（推荐）或 JPG
- **尺寸**：建议宽度 1200-1600px
- **内容**：
  - screenshot-1：展示主界面、代码高亮、数学公式
  - screenshot-2：展示 Mermaid 图表功能
